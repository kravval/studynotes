# StudyNotes



Персональная система управления IT-заметками.



## О проекте



Приложение для работы с заметками из Obsidian: импорт, поиск, удобный просмотр с рендерингом Markdown.



## Стек



- Java 17

- Spring Boot

- Spring Data JPA

- PostgreSQL

- Thymeleaf

- commonmark-java



## Статус



В разработке.
