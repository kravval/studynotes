\# StudyNotes



Персональная система управления IT-заметками.



\## Что это



Приложение для работы с заметками из Obsidian: импорт, поиск,

удобный просмотр с рендерингом Markdown.



\## Стек



\- Java 17

\- Spring Boot

\- PostgreSQL

\- Thymeleaf
